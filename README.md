# ShoppingList

Xamarin.Forms goodlooking UI sample using the new SwipeView. Based on [Shopping List design](https://dribbble.com/shots/2340386-Shopping-List) by Hila Peleg.

The sample is available for two platforms:

- iOS
- Android

Features:
- Bindable Layout.
- Expandable Control.
- PancakeView.
- Triggers.
